**HomieGo** is a user-friendly platform designed to make the search for hostels and PGs hassle-free. Built using **NextJs**, the platform is optimized for lightning-fast performance and exceptional SEO, achieving **Top 5 Google search rankings**.

## Features

- **Hostel and PG Listings**: Browse and compare a variety of hostels and PGs.
- **Advanced Search Filters**: Find the perfect accommodation based on location, price, and amenities.
- **User Reviews**: Read honest feedback to make informed decisions.
- **Interactive Map Integration**: View accommodations on a map for easy navigation.
- **Responsive Design**: Enjoy a seamless experience across all devices.

## Technologies

- **NextJs**: A React framework for building high-performance web applications.
- **Tailwind CSS**: A utility-first CSS framework for designing responsive and interactive layouts.
- **PostgreSQL**: A robust database for managing listing data.
- **Vercel**: A reliable platform for deploying and hosting web applications.
